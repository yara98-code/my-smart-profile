مشروع الملف الشخصي الذكي - My Smart Profile
نبذة عن المشروع:

مشروع "الملف الشخصي الذكي" هو موقع ويب شخصي تفاعلي يتكون من عدة صفحات، مصمم باستخدام HTML وCSS وJavaScript وjQuery وBootstrap، ويهدف إلى عرض معلومات المستخدم بأسلوب عصري وذكي. يتضمن الموقع صفحات متعددة مثل الصفحة الرئيسية، من أنا، معرض الأعمال، التواصل، صفحة المرح، وصفحة التفاعل الذكي.
مكونات المشروع:
1. index.html - الصفحة الرئيسية:

تحتوي على ترحيب بالمستخدم، وصورة شخصية، وروابط موجهة لباقي الصفحات.
2. about.html - من أنا:

تعرض نبذة تعريفية عن الشخص، صور ومحتوى منسق، مع زر "عرض المزيد" لإظهار محتوى إضافي.
3. portfolio.html - معرض الأعمال:

تتضمن نماذج من أعمال أو مشاريع الشخص مع تأثيرات بصرية على العناصر.
4. contact.html - صفحة التواصل:

نموذج تواصل بسيط يحتوي على حقول الاسم والبريد والرسالة.
5. fun.html - صفحة المرح:

تفاعلية تتضمن:

    زر لإظهار/إخفاء رسالة مرحة.
    زر لعرض صورة بتأثير fadeToggle.
    زر لتحميل اقتباس أو نكتة من ملف خارجي (quote.txt).

6. smart.html - صفحة التفاعل الذكي:

تتضمن:

    إدخال الاسم وعرض ترحيب حي باستخدام jQuery.
    تغيير لون الخلفية عند تمرير الفأرة.
    زر لتحريك صندوق بطريقة تفاعلية.
    مفتاح لتبديل الوضع الليلي (Dark Mode).

التقنيات المستخدمة:

    HTML5 / CSS3
    Bootstrap 5.3
    JavaScript
    jQuery

ملفات داعمة:

    style.css: يحتوي على التنسيق العام للموقع.
    quotes.js: يحتوي على مصفوفة من الاقتباسات أو النكت العشوائية.
    quote.txt: ملف نصي خارجي يتم تحميل محتواه في صفحة المرح.

التحسينات المقترحة:

    إضافة مؤثرات بصرية متقدمة باستخدام مكتبة مثل AOS أو Animate.css.
    إضافة LocalStorage لحفظ اسم المستخدم وتذكره بين الزيارات.
    تحسين تجربة المستخدم في الوضع الليلي بإضافة تبديل الألوان للعناصر بشكل أدق.
    فصل JavaScript في ملفات مستقلة لتحسين التنظيم.
    إضافة فوتر موحد لكل الصفحات.
    ربط نموذج التواصل بخدمة خارجية مثل Formspree أو Google Forms.
    إضافة صفحة "الشهادات" أو "العملاء" لعرض آراء المستخدمين.
